# Predictive Analytics Using Historical Data

This project builds a predictive model using historical monthly sales data to forecast future sales trends.

## Files
- `predictive_analytics.ipynb` – Notebook with data preprocessing, model building, evaluation, and visualization
- `sales_data.csv` – Dataset used for analysis
- `report.pdf` – Short report with results and conclusions

## Model Used
Linear Regression

## Evaluation Metrics
MAE, MSE, RMSE

## Outcome
The model forecasts future sales trends based on historical monthly data and visualizes actual vs predicted values.
